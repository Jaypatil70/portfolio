## 🛠️ Built With

This project relies on modern web technologies and libraries to create an interactive and performant user experience:
- **HTML5 & CSS3**: Core structure and responsive styling.
- **JavaScript (ES6+)**: Interactive features and smooth functioning.
- **[Three.js](https://threejs.org/) & [Vanta.js](https://www.vantajs.com/)**: Dynamic 3D network-style interactive backgrounds.
- **[GSAP](https://gsap.com/) & ScrollTrigger**: High-performance scroll-based animations.
- **Google Fonts**: Custom typography using 'Outfit' font.
- **Font Awesome**: Vector icons for UI elements.

## 🌟 Features

- **Interactive 3D Background**: A visually appealing Vanta.js animated network background reflecting my background in networking and cloud infrastructure.
- **Smooth Scroll Animations**: Powered by GSAP and ScrollTrigger for revealing content elegantly as the user scrolls.
- **Responsive Design**: Designed to work flawlessly across all devices—desktops, tablets, and smartphones.
- **Projects Showcase**: Highlights the Salesforce automation and data modeling systems I've built.
- **Easy Navigation**: Fixed glassmorphism navigation bar for quick access to different sections.

## ⚙️ Getting Started

To view the website locally, you don't need to install any heavy dependencies or build tools.

1. Clone the repository:
   ```bash
   git clone https://github.com/Jaypatil70/your-repo-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```
3. Simply open `index.html` in your favorite web browser:
   - Double-click `index.html` using your file explorer, OR
   - If you use VS Code, you can use the **Live Server** extension for hot-reloading.
